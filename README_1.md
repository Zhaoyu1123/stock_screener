# 高胜率选股策略 · 精准狙击优化版
> 核心思路：不是每天都交易，只在最好的时机出手。
> 股市有风险，本策略仅供参考，不构成投资建议。
---
## ⚠️ 关于"每天都有收益"
- A 股 T+1 限制，今天买明天才能卖
- 正确目标：**每周盈利 3 天以上（胜率 60%+）**，单次盈亏比 ≥ 2:1
- "每天盈利"的策略要么是回测过拟合，要么是骗局
---
## 优化一：市场情绪过滤器（开盘前必做）
### ✅ 可以交易
- 昨日涨停板数量 > 50 只
- 今日高开股票数量 > 低开
- 沪指站在 5 日均线之上
### ❌ 今天空仓
- 昨日涨停板数量 < 20 只
- 大盘跌破 20 日均线
- 重大政策 / 财报发布日前后
---
## 优化二：「强者恒强」精选法
在原策略基础上，额外满足：
- 近 5 日涨幅 `> 10%`（已在强势上涨中）
- 近 5 日最大回撤 `< 3%`（涨得稳）
- 今日涨幅排名全市场前 `5%`
- 所在板块是近 3 日涨幅最大的前 3 个之一
---
## 优化三：「最佳买点」时间窗口
| 时间 | 操作 |
|------|------|
| 09:30~09:45 | 开盘博弈，不买 |
| 09:45~10:30 | 第一波拉升，观察 |
| 10:30~11:30 | 主力洗盘，不买 |
| **14:00~14:30** | ⭐ **最佳买入窗口** |
| 14:30~15:00 | 确认收盘走势，可加仓 |
> 只在 14:00~14:30 买入，避开早盘波动，胜率提升约 15%。
---
## 优化四：「阶梯止盈」
| 涨幅 | 操作 |
|------|------|
| +3% | 卖出 20%，回本保底 |
| +5% | 再卖 30%，锁定利润 |
| +8% | 再卖 30%，剩 20% 博涨停 |
| 涨停 | 持有到次日，看高开低走还是继续 |
---
## 优化五：「连板跟踪」策略
### 买入条件
- 昨日涨停（第一板）
- 今日高开 2%+ 且未封死涨停
- 涨停板封板时间 > 14:00
- 板块有 2 只以上同日涨停
### 止损 / 止盈
- 止损：今日收盘跌破昨日涨停价
- 止盈：次日高开 3%+ 直接卖出
---
## 优化六：「不操作」检查清单
满足其一，今天空仓：
- 大盘跌超 1%
- 北向资金净流出超 30 亿
- 昨天已经亏损（冷静一天）
- 没有满足全部条件的标的
- 持仓已有浮亏超 5%（先处理亏损仓位）

## 优化后预期表现
| 指标 | 原策略 | 优化后 |
|------|--------|--------|
| 每周交易天数 | 5 天 | **2~3 天** |
| 单次胜率 | 65% | **75%~80%** |
| 平均盈亏比 | 1.5:1 | **2.5:1** |
| 月度盈利概率 | 70% | **85%+** |
---
## 核心原则
> **亏的时候少亏，赚的时候多赚，不好的时候不交易。**
> 交易次数少了，每次质量更高，总收益反而更好。

# K线基础教程

> 看懂K线是使用选股策略的前提。本节覆盖K线周期、均线系统、买卖点判断。

---

## 一、K线周期

每种周期代表**一根K线（蜡烛）所覆盖的时间长度**：

| 周期 | 一根K线 = | 适合看什么 |
|------|-----------|-----------|
| **分时** | 当天逐分钟走势（折线图） | 盘中买卖点、当日强弱 |
| **五日** | 最近5个交易日的分时拼接 | 短线连续几天的走势节奏 |
| **日K** | 1个交易日 | 短线操作（1~2周持股） |
| **周K** | 1周（周一~周五合并成一根） | 中线趋势（1~3个月） |
| **月K** | 1个月 | 中长线趋势（半年~1年） |
| **年K** | 1年 | 长期走势、历史大底大顶 |

### 一根K线包含的信息

```
  │  ← 上影线（最高价）
  │
┌─┴─┐
│   │ ← 实体（开盘价 ↔ 收盘价）
│   │    红色：收盘 > 开盘（涨）
│   │    绿色：收盘 < 开盘（跌）
└─┬─┘
  │
  │  ← 下影线（最低价）
```

---

## 二、均线系统（MA）

### MA 是什么

MA = Moving Average（移动平均线），MA后面的数字 = 取最近多少根K线的收盘价平均值。

**本质：MA就是"最近X天大家买入的平均成本"，股价在均线上方说明大家赚钱（多头强），在下方说明大家亏钱（空头强）。**

| 均线 | 含义 | 代表 |
|------|------|------|
| MA5 | 最近 5 根K线的平均收盘价 | 一周的平均成本 |
| MA10 | 最近 10 根K线的平均收盘价 | 两周的平均成本 |
| MA20 | 最近 20 根K线的平均收盘价 | 一个月的平均成本 |
| MA60 | 最近 60 根K线的平均收盘价 | 一个季度的平均成本 |

### 计算举例

```
某股最近5天收盘价: 10.0, 10.2, 10.5, 10.3, 10.5
MA5 = (10.0 + 10.2 + 10.5 + 10.3 + 10.5) / 5 = 10.3
```

### 不同K线周期下均线代表的时间跨度

| 均线 | 日K上 | 周K上 | 月K上 |
|------|-------|-------|-------|
| MA5 | 5天 | 5周(1个月) | 5个月 |
| MA10 | 10天(2周) | 10周(2个月) | 10个月 |
| MA20 | 20天(1个月) | 20周(5个月) | 20个月 |
| MA60 | 60天(3个月) | 60周(1年多) | 60个月(5年) |

### 均线的作用

1. **判断趋势方向** — 均线排列方式
2. **当支撑/压力** — 上涨趋势中回调碰MA反弹；下跌趋势中反弹碰MA被压回
3. **判断强弱** — 股价在均线上方=强，下方=弱

---

## 三、多头排列与空头排列

### 多头排列（上涨趋势确认）

```
MA5 > MA10 > MA20 > MA60

短期均线在最上面，长期均线在最下面。
所有周期的持股者都在赚钱，趋势向上。

示例:
  MA5:  3.80  ← 最大（最上面）
  MA10: 3.70
  MA20: 3.60
  MA60: 3.50  ← 最小（最下面）
```

### 空头排列（下跌趋势）

```
MA5 < MA10 < MA20 < MA60

短期均线在最下面，长期均线在最上面。
越近期买的人亏得越多。

示例:
  MA5:  3.43  ← 最小（最下面）
  MA10: 3.45
  MA20: 3.64
  MA60: 3.70  ← 最大（最上面）
```

### 从空头到多头的演变过程

```
第1阶段: 股价突破，但均线仍是空头排列
  价格 > MA60 > MA20 > MA10 > MA5

第2阶段: MA5 拐头上穿 MA10（开始扭转）
  价格 > MA60 > MA20 > MA5 > MA10

第3阶段: 均线收拢、交叉

第4阶段: 多头排列完成 ✅
  价格 > MA5 > MA10 > MA20 > MA60
```

> 股价是先行指标，均线是滞后指标。股价先突破，均线需要时间跟上。
> 一般从空头转为多头需要 **3~6周**。

## 四、买卖点判断实战

### 短线（持股1~5天）：日K + 分时

#### 买点

```
日K连续5天窄幅震荡（箱体）:
  3.40 ───────────── 箱顶
  │  小阴 小阳 十字 小阳 │
  3.20 ───────────── 箱底

第6天: 一根大阳线 + 成交量翻倍 → 突破箱顶
  收盘 3.55, 量是前5日均量的2倍
```

确认步骤：
1. 日K看到突破信号后不急着追
2. 切到**分时图**，等 14:00~14:45 尾盘确认不回落
3. 尾盘价格仍在箱顶上方 → 买入

#### 卖点

| 信号 | 操作 |
|------|------|
| 次日高开后分时持续走低，跌破均价线 | 卖出 |
| 日K出现长上影线（冲高回落） | 减仓 |
| 跌破突破当天最低价 | 止损走人 |

**核心原则**：短线看的是当下的强弱，一旦转弱立刻走。

---

### 波段（持股2周~2个月）：周K定方向，日K找时机

#### 买点

```
周K走势:
  连跌数月后企稳...
  MA5 拐头向上 → 穿过MA10 → 穿过MA20 → 多头排列形成

切到日K找入场:
  股价跟随周线方向上涨后，回踩到日线MA10或MA20附近
  回踩不破 + 缩量 → 某天止跌收阳 = 波段买点
```

#### 卖点

| 信号 | 操作 |
|------|------|
| 周K收出带长上影的阴线 | 减仓50% |
| 周K跌破MA5且MA5拐头向下 | 全部卖出 |
| 涨幅达到15~20% | 至少止盈一半 |
| 日K连续3天缩量创新高（量价背离） | 警惕，准备卖 |

**核心原则**：波段看的是周线趋势有没有走坏，只要周线多头排列没被破坏就拿着。

---

### 大趋势判断（牛市 vs 熊市）：月K

#### 牛市特征

```
月K:
  MA5 > MA10 > MA20 > MA60（月线多头排列）
  股价在所有月均线之上
  每次回调不破MA10
  → 做多
```

#### 熊市特征

```
月K:
  MA5 < MA10 < MA20 < MA60（月线空头排列）
  股价在所有月均线之下
  每次反弹碰到MA10就被打下来
  → 空仓或极轻仓
```

#### 震荡市特征

```
月K:
  均线缠绕在一起，方向不明
  股价围绕均线上下穿越
  → 不适合趋势策略，空仓或只做短线
```

---

## 五、快速判断口诀

| 问题 | 看什么 | 怎么判断 |
|------|--------|----------|
| 现在是牛还是熊？ | 上证月K | 月线 MA5/10/20 排列方向 |
| 这只股能不能做波段？ | 个股周K | 周线均线是否多头排列 |
| 明天能不能买？ | 个股日K + 分时 | 日K有信号 + 尾盘确认 |

**总结：大周期定方向（月K判牛熊）→ 中周期选股（周K找趋势向上的）→ 小周期择时（日K/分时找具体买卖点）。永远顺着大周期的方向做，不要在熊市里做多。**
