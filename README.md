# 高胜率选股策略

> **免责声明：** 本策略仅供参考，不构成投资建议。股市有风险，投资需谨慎。

---

## 一、核心逻辑

**不追求"选对"，而是追求"在最强的时机买最强的票"。**

胜率高的本质不是预测，是筛选出**已经在涨、还在加速**的标的，让市场替你做验证。

---

## 二、策略一：「龙头共振」策略

> 适用场景：日内 + 隔夜持有  
> 核心：板块 + 个股双重共振，买强不买弱

### 第一层：市场环境过滤

- 当日沪指涨幅 `> 0%`（大盘不跌）
- 市场上涨家数 `>` 下跌家数（多数股票在涨） 
- 北向资金当日净流入 `> 0`（外资认可）

### 第二层：板块共振

- 所在板块当日涨幅 `> 1.5%`（板块整体在动）
- 板块内涨停 / 涨幅 `>5%` 的个股 `≥ 3` 只（不是孤立的一只）

### 第三层：个股精选

- 股价位于近 60 日内高点附近（突破或在创新高路上）
- 今日成交量 > 20 日均量的 1.5 倍（量能放大，主力介入）
- 今日涨幅 3%~8%（涨了但没涨停，还有空间）
- 均线多头排列：MA5 > MA10 > MA20 > MA60
- 非 ST、非科创板、非创业板
- 流通市值 50亿~500亿（主力容易操控，机构也感兴趣）
- PE 在 15~60 之间（有业绩支撑，但不过分高估）

| 条件 | 要求 |
|------|------|
| 股价位置 | 近 60 日高点附近（突破或创新高路上） |
| 成交量 | 今日 `>` 20日均量 `× 1.5` |
| 今日涨幅 | `3% ~ 8%` |
| 均线排列 | MA5 `>` MA10 `>` MA20 `>` MA60 |
| 板块 | 非 ST、非科创板、非创业板 |
| 流通市值 | `30亿 ~ 300亿` |
| PE | `15 ~ 60` |

### 第四层：主力资金验证

- 主力净流入 `> 5000 万`（当日）
- 龙虎榜席位以机构 / 知名游资为主

---

## 三、策略二：「启动信号」策略

> 核心：在股票从盘整**突破的第一天**买入

### 信号条件

1. 股价横盘整理 `≥ 15` 个交易日
2. 今日放量突破箱体上沿
   - 成交量 `>` 前15日均量 `× 2`
   - 收盘价 `>` 近15日最高价
3. 当日涨幅 `3% ~ 7%` （突破有力但未涨停）
4. MACD 金叉 or DIF 从负转正
5. 主力净流入 > 5000 万
6. 公司近期有催化剂（业绩预增、行业政策、并购等）


## 四、买入执行规则

| 规则 | 内容 |
|------|------|
| 买入时机 | 14:00~14:45，确认尾盘不跌则买入 |
| 单只仓位 | ≤ 总仓位 **20%** |
| 分批操作 | 先买 50%，次日确认再加仓 |

---

## 五、止损规则（胜率高的关键）

> **胜率高 ≠ 每次都赚，而是亏的时候亏得少。**

| 触发条件 | 操作 |
|---------|------|
| 跌破当日最低价 | ❌ 立即止损，无条件 |
| 次日低开 > 3% | ❌ 止损 |
| 板块当日跌 > 1% | ❌ 止损 |
| 持有 3 日未涨 | ❌ 止损或减仓 |

---

## 六、止盈规则

| 情况 | 操作 |
|------|------|
| 涨幅 > 10% | ✅ 卖出 50% |
| 涨停次日高开低走 | ✅ 卖出 |
| 缩量上涨 | ⚠️ 减仓 |
| 大盘转弱 | ✅ 无条件减仓 |

### 🎓 为什么这个策略胜率高？
1. **三重过滤**：大盘 → 板块 → 个股，只在最强的环境下买最强的票
2. **量价验证**：放量突破是主力意图最明确的信号
3. **铁纪止损**：控制亏损次数和金额，胜率自然上升
4. **不摸鱼**：不在弱市强买，不在震荡市操作

---

### ⚠️ 重要说明
- 这套策略在**牛市/偏多行情**下胜率能达到 70~85%
- 在熊市或震荡市，胜率会下降，**此时最佳策略是空仓**
- 所谓"90% 胜率"通常需要加上**"在符合条件时才交易"**的前提
- 历史胜率不代表未来，每次都要严格执行止损

## 七、脚本使用方法

### 环境准备

```bash
# 创建虚拟环境并安装依赖
python3 -m venv .venv
.venv/bin/pip install akshare pandas numpy
```

### 运行选股

```bash
# 默认严格模式 (沪指>0%, 涨多跌少, 北向>0)
.venv/bin/python3 stock_screener.py

# 完全跳过市场环境检查，强制执行两套策略选股
.venv/bin/python3 stock_screener.py --skip-market-check
```

### 第一层参数调节

| 参数 | 说明 | 默认值 | 示例 |
|------|------|--------|------|
| `--min-sh-pct` | 沪指最低涨幅要求 (%) | `0.0` | `--min-sh-pct -0.5` 允许微跌 |
| `--no-up-majority` | 跳过「涨多跌少」检查 | 不跳过 | `--no-up-majority` |
| `--min-north` | 北向资金最低净流入 (亿元) | `0` | `--min-north -10` 允许流出10亿 |
| `--skip-market-check` | 跳过整个第一层检查 | 不跳过 | `--skip-market-check` |

### 组合示例

```bash
# 放宽沪指到允许微跌0.3%，且不要求涨多跌少
.venv/bin/python3 stock_screener.py --min-sh-pct -0.3 --no-up-majority

# 仅放宽北向资金要求（允许流出5亿以内）
.venv/bin/python3 stock_screener.py --min-north -5

# 全部放宽
.venv/bin/python3 stock_screener.py --min-sh-pct -0.5 --no-up-majority --min-north -10
```
